# ollama-batch-api

Drop-in replacement for Anthropic and OpenAI batch APIs — self-hosted, running against local Ollama.

## Installation

```bash
composer require digitaldreams/ollama-batch-api
```

## API Endpoints

- `POST /api/anthropic/v1/messages/batches` — Anthropic Message Batches
- `POST /api/openai/v1/batches` — OpenAI Batch API
- `POST /api/openai/v1/files` — OpenAI Files API

See [documentation](https://github.com/digitaldreams/ollama-batch-api) for full usage.
